HƯỚNG DẪN DEPLOY FRONTEND NEXT.JS LÊN VERCEL (MIỄN PHÍ VÀ SIÊU TỐC)
===================================================================

Vercel là nền tảng tốt nhất thế giới để chạy Next.js (bởi vì họ chính là công ty tạo ra Next.js).
Nó hoàn toàn miễn phí và KHÔNG BAO GIỜ bị ngủ đông (sleep) như Render.

LƯU Ý TRƯỚC KHI BẮT ĐẦU: 
Bạn phải chắc chắn Backend trên Render đã chạy thành công và lấy được cái Link API (ví dụ: `https://booking-system-backend-api.onrender.com`).

BƯỚC 1: KẾT NỐI GITHUB VỚI VERCEL
-----------------------------------
1. Truy cập vào trang web: https://vercel.com/
2. Đăng nhập bằng tài khoản GitHub (Sign in with GitHub).
3. Ở màn hình chính, nhấn nút "Add New" (Thêm mới) -> Chọn "Project" (Dự án).
4. Vercel sẽ hiện ra danh sách các kho GitHub của bạn. Tìm kho `booking_system_deploy_test` và nhấn nút "Import".

BƯỚC 2: CẤU HÌNH DỰ ÁN FRONTEND
-----------------------------------
Màn hình cấu hình (Configure Project) sẽ hiện ra. Bạn cần chỉnh ĐÚNG CHUẨN các mục sau:

- Project Name: Đặt tên tùy ý (VD: `booking-system-web`). Tên này sẽ làm thành cái link truy cập web của bạn (VD: booking-system-web.vercel.app).
- Framework Preset: Vercel sẽ tự động nhận diện là `Next.js`. Bạn để nguyên.
- Root Directory: (CỰC KỲ QUAN TRỌNG) Bạn nhấn nút "Edit" và chọn thư mục `frontend/web`. (Vì code Next.js của bạn nằm sâu ở trong thư mục này).

BƯỚC 3: CẤU HÌNH BIẾN MÔI TRƯỜNG (ENVIRONMENT VARIABLES)
-----------------------------------------------------------
Nhấn vào mục "Environment Variables" để mở rộng nó ra.
Ở đây bạn cần nhập 2 biến để Frontend biết đường tìm đến cái Backend trên Render của bạn.

1. Biến kết nối API:
   - Name: `NEXT_PUBLIC_API_URL`
   - Value: `https://<ten-render-cua-ban>.onrender.com/api/v1/web` 
     (Thay `<ten-render-cua-ban>` bằng link thật mà Render cấp cho bạn nhé. Đừng quên cái đuôi /api/v1/web)

2. Biến kết nối Socket (Realtime chọn ghế):
   - Name: `NEXT_PUBLIC_SOCKET_URL`
   - Value: `https://<ten-render-cua-ban>.onrender.com` 
     (Chỉ lấy domain chính, không có đuôi đằng sau).

Sau khi điền xong cả 2 biến, bạn nhấn nút "Add" cho từng biến để lưu lại.

BƯỚC 4: DEPLOY VÀ TẬN HƯỞNG
-----------------------------------
- Nhấn nút "Deploy" to đùng màu đen ở dưới cùng.
- Vercel sẽ tự động build code. Mất khoảng 1-2 phút, lúc này nó sẽ bắn pháo hoa lên màn hình báo hiệu thành công 🎉.
- Bấm vào hình cái cửa sổ Web để truy cập ngay vào dự án của bạn!

TỪ BÂY GIỜ TRỞ ĐI:
Mỗi khi bạn sửa code trên máy tính, bạn chỉ cần gõ lệnh đẩy lên GitHub (`git push`).
CẢ Render và Vercel sẽ tự động phát hiện code mới và tự động cập nhật lại web cho bạn. Bạn không cần làm thủ công nữa!
